### Сессии
```php
<?php
 session_start();
 $_SESSION['test'] = 'hello';
?>
<?php
 session_start();
 echo $_SESSION['test']; 
?>
```

### Возможные проблемы при работе с сессиями 
```php
<?php
 session_start();
?>

<?php
 session_start();
<p></p>
?>

<?php
 session_start();
 echo 'text';
?>
```

### Использование сессий на одной странице
```php
<?php
 session_start();
 if (!isset($_SESSION['start_time'])) {
    $_SESSION['start_time'] = time();
 } else {
  echo time() - $_SESSION['start_time'];
 }
?>
```

### Удаление сессий
```php
<?php
 session_start();
 
 if (!isset($_SESSION['counter'])) {
  $_SESSION['counter'] = 1;
 } else {
  $_SESSION['counter']++;
 }
 if ($_SESSION['counter'] >= 10) {
  echo $_SESSION['counter'];
 }
?>
```

### Завершение всей сессии 
```php
<?php
 session_destroy();
?>
```

### Сессии и формы
```php
<form method="GET">
 <input name="name">
 <input name="surname">
 <input name="age">
 <input type="submit">
</form>

<?php
 session_start();
 
 if (!empty($_GET)) {
  $_SESSION['name'] 
   = $_GET['name']; 
  $_SESSION['surname'] 
   = $_GET['surname']; 
  $_SESSION['age'] 
   = $_GET['age']; 
 }
?>
<?php
session_start();
 if (!empty($_SESSION)) {
  echo $_SESSION['name'] . ' ' . $_SESSION['surname'] . ' ' . $_SESSION['age']; 
 }
?>
```

### Запись массивов в сессии
```php
<form method="GET">
 <input name="name">
 <input name="surname">
 <input name="age">
 <input name="salary">
 <input name="whatsup">
 <input type="submit">
</form>

<?php
session_start();
 if (!empty($_GET)) {
  $_SESSION['arr'] = $_GET;
  var_dump(['arr']);
 }
?>

<?php
session_start();
 if (!empty($_SESSION)) {
  $arr = $_SESSION['arr'];
  echo "<ul>";
  foreach($arr as $elem){
  echo "<li> $elem </li>";
  }
  echo "</ul>";
 }
?>
```
