### Вставка переменных в строки
```php
<?php
$name = "user";
echo "hello, " . $name . "!";
?>
```

### Вставка элементов массива
```php
<?php
$arr = ['1', '2', '3'];
echo "aaa " . $arr[0]
        . " bbb "
        . $arr[1];
?>
```

### Вставка элементов ассоциативных массивов
```php
<?php
$arr = ['a' => 1, 'b' => 2, 'c' => 3];
echo "text {$arr['a']} text {$arr['b']}";
?>
```

### Цикл и вставка переменных
```php
<?php
for ($i = 1; $i <= 10; $i++) {
    for ($j = 1; $j <= 10; $j++) {
        echo "nums: " . $i
                . " " . $j
                . "<br>";
    }
}
?>
```

### Вставка элементов массивов в цикле
```php
<?php
$arr = ['a' => 1, 'b' => 2, 'c' => 3];
foreach ($arr as $key => $elem) {
    echo "pair: " . $elem . " " . $key . "<br>";
}
?>
```

### Генерация HTML таблицы с помощью двух вложенных циклов
```php
<?php
$products = [
        [
                'name'   => 'product1',
                'price'  => 100,
                'amount' => 5,
        ],
        [
                'name'   => 'product2',
                'price'  => 200,
                'amount' => 6,
        ],
        [
                'name'   => 'product3',
                'price'  => 300,
                'amount' => 7,
        ],
];
echo '<table>';
foreach ($products as $product) {
    echo '<tr>';
    foreach ($product as $cell) {
        echo "<td>$cell</td>";
    }
    echo '</tr>';
}
echo '</table>';
?>
```

### Условия и разрыв PHP кода

### Блок elseif в условиях для разрыва PHP кода
```php
<?php
$test = true;
?>
<?php if ($test === true): ?>
    <div>
        <p>text1</p>
        <p>text1</p>
        <p>text1</p>
    </div>
<?php elseif ($test === false): ?>
    <div>
        <p>text2</p>
        <p>text2</p>
        <p>text2</p>
    </div>
<?php else: ?>
    <div>
        <p>text-</p>
        <p>text-</p>
        <p>text-</p>
    </div>
<?php endif; ?>
```
