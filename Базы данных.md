```php
<?php

ini_set('display_startup_errors', 1);
error_reporting(E_ALL);

$host = 'ct439599.tw1.ru';
$user = 'ct439599_test';
$pass = 'uQ7TRxQR';
$name = 'ct439599_test';

$link = mysqli_connect($host, $user, $pass, $name);

mysqli_query($link, "SET NAMES 
		'utf8'");
?>
```


### Тестирование работоспособности базы данных
