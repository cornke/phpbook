### Глобальные переменные
```php
<?php
$num = 1;
function func()
{
    global $num;
    return $num++;
}
func();
echo $num; // должно вывести 2
?>

<?php
$num = 1;
function func()
{
    global $num;
    $num++;
    return $num;
}
echo func(); // должно вывести 2
?>

<?php
$num = 1;

function func()
{
    global $num;
    return $num;
}

echo func(); // должно вывести 1
?>
```

### Параметры-ссылки
```php
<?php
$num = 1;
function func(&$num)
{
    $num++;
}
func($num);
echo $num; // должно вывести 2
?>

<?php
$arr = [1, 2, 3, 4, 5];
function func(&$arr)
{
    $arr[0] = '!';
}
func($arr);
var_dump($arr);
?>
```
