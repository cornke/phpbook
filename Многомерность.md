### Перебор многомерных массивов
```php
<?php
 $arr = [[1, 2, 3], [4, 5, 6, 7], [8, 9]];
$res = 0;
foreach ($arr as $sub) {
  foreach ($sub as $elem) {
   $res += $elem;
  }
}
echo $res;
?>
```

### Перебор трехмерного массива 
```php
<?php

  [
   [1, 2, 3],
   [6, 7, 8],
   [3, 8, 4],
   [6, 7, 9],
  ],
  [
   [9, 1, 2],
   [4, 5, 6],
  ],
  [
   [9, 1, 2],
   [4, 5, 6],
   [5, 6, 3],
  ],
 ];
$res = 0;
foreach ($arr as $sub) {
  foreach ($sub as $subsub) {
   foreach ($subsub as $elem) {
    $res += $elem;
   }
  }
 }
echo $res;
?>
```
