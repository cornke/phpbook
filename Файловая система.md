### Запись файлов
```php
<?php
$arr = [4, 5, 9, 1];
$res = 0;
foreach ($arr as $elem) {
    $res += $elem;
}
file_put_contents('./sum.txt', $res);
?>
```

### Комбинация операций чтения и записи файла
```php
<?php
$num = file_get_contents('2.txt');
$sqrt = $num ** 2;
file_put_contents('2.txt', $sqrt);
?>

<?php
$num = file_get_contents('count.txt');
file_put_contents('count.txt', ++$num);
?>

<?php
$arr = ['1.txt', '2.txt', '3.txt'];
$str = '';
foreach ($arr as $elem) {
    $str .= file_get_contents($elem);
}
file_put_contents('./new.txt', $str);
?>
```

### Разбираем содержимое папки
```php
<?php
$dir = 'dir';
$files = array_diff(scandir($dir),
        ['..', '.']);
foreach ($files as $file) {
    $path = $dir. '/' . $dir;
    if (is_dir($path)) {
        echo file_get_contents($path);
    }
}
?>

<?php
$dir = 'dir';
$files = array_diff(scandir($dir),
        ['..', '.']);
foreach ($files as $file) {
    $path = $dir. '/' . $file;
    if (is_file($path)) {
        echo file_get_contents($path);
    }
}
?>

<?php
$path = 'dir/';
$files = array_diff(scandir($path), ['.', '..']);

foreach ($files as $file) {
    $fullPath = $path . $file;
    if (is_file($fullPath)) {
        $time = "\n" . date('Y-m-d H:i:s');
        file_put_contents($fullPath, $time, FILE_APPEND);
    }
}
?>
```

### Подключение файлов

### Однократное подключение файлов
```php

    return $num ** 2;
}
?>
<?php

    return $num ** 3;
}
?>
<?php

?>
```
