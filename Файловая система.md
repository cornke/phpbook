### Чтение файлов
```php
<?php
echo file_get_contents('1.txt') + file_get_contents('2.txt');
?>
```

### Запись файлов
```php
<?php
$arr = [4, 5, 9, 1];
$res = 0;
foreach ($arr as $elem) {
    $res += $elem;
}
file_put_contents('./sum.txt', $res);
?>
```

### Комбинация операций чтения и записи файла
```php
<?php
$num = file_get_contents('2.txt');
$sqrt = $num ** 2;
file_put_contents('2.txt', $sqrt);
?>

<?php
$num = file_get_contents('count.txt');
file_put_contents('count.txt', ++$num);
?>

<?php
$arr = ['1.txt', '2.txt', '3.txt'];
$str = '';
foreach ($arr as $elem) {
    $str .= file_get_contents($elem);
}
file_put_contents('./new.txt', $str);
?>
```

### Относительные пути
```php
<?php
echo file_get_contents('dir1/dir2/test.txt');
?>

<?php
echo file_get_contents('../dir1/dir2/test.txt');
?>

<?php
echo file_get_contents('../../dir/test.txt');
?>

<?php
echo file_get_contents('../../../../dir1/dir2/dir3/test.txt');
?>
```

### Абсолютные пути к папке и файлу скрипта
```php
<?php
echo file_get_contents('../../__FILE__');
?>

<?php
echo file_get_contents('../../../__FILE__');
?>
```

### Переименовывание файлов
```php
<?php
rename('old.txt', 'new.txt');
?>
```

### Перемещение файлов
```php
<?php
rename('file.txt', 'dir/file.txt');
?>

<?php
rename('dir1/file.txt', 'dir2/file.txt');
?>
```

### Перемещение файлов
```php
<?php
for ($i = 0; $i <= 5 ; $i++) {
    copy('new.txt', "$i.txt");
}
?>
```

### Удаление файлов
```php
<?php
$arr = ['1.txt', '2.txt', '3.txt'];
foreach ($arr as $elem) {
    unlink($elem);
}
file_put_contents('./new.txt', $str);
?>
```

### Определение размера файлов
```php
<?php
echo filesize('meow.txt');
?>

<?php
echo filesize('meow.txt') / 1024;
?>

<?php
echo filesize('meow.jpeg') / 1024 / 1024;
?>

<?php
echo filesize('mov.mp4') / 1024 / 1024 / 1024;
?>
```

### Проверка существования файлов
```php
<?php
var_dump(file_exists('file.txt'));
?>

<?php
if (!(file_exists('file.txt'))) {
    file_put_contents('./sum.txt', '!');
}
?>

<?php
if (file_exists('message.txt')) {
    echo file_get_contents('message.txt');
}
?>
```

<?php
