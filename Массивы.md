###Как посмотреть содержимое массива
```php
<?php
$a = ['a', 'b', 'c'];
echo $a;
var_dump($a);
?>
```

###Отдельный элемент массива
```php
<?php
$arr = ['a', 'b', 'c'];
echo $arr[0];
echo $arr[1];
echo $arr[2];
?>

<?php
$arr = ['a', 'b', 'c', 'd', 'e'];
echo $arr[0] . '+' . $arr[1] . '+' . $arr[2] . '+' . $arr[3];
?>

<?php
$arr = [1, 2, 3, 4, 5];
echo $arr[0] + $arr[1] + $arr[2] + $arr[3] + $arr[4];
?>

<?php
$arr = [2, 5, 3, 9];
$res = ($arr[0] * $arr [1]) + ($arr[2] * $arr[3]);
echo $res;
?>
```

###Длина массива
```php
<?php
$arr = ['a', 'b', 'c', 'd', 'e'];
echo count($arr);
?>

<?php
$arr = ['a' => 1, 'b' => 2, 'c' => 3, 'd' => 4];
echo count($arr);
?>
```

###Изменение элементов массива
```php
<?php
$arr = ['a' => 1, 'b' => 2, 'c' => 3];
$arr['a'] = 7;
$arr['b'] = 8;
$arr['c'] = 9;
?>
```

###Перезапись элементов массива
```php
<?php
$arr = ['a' => 1, 'b' => 2, 'c' => 3];
$arr['a'] += 1;
$arr['b'] += 1;
$arr['c'] += 1;
var_dump($arr);
?>
```

###Заполнение массивов

```php
<?php
$arr = [];
$arr[] = 1;

$arr[] = 3;
$arr[] = 4;
$arr[] = 5;
?>

<?php
$arr = [1, 2, 3];

```

###Явное указание ключей
```php
<?php
$arr = [];
$arr['year'] = '2026';
$arr['month'] = '02';

?>
```

###Ключи из переменных
```php
<?php
$arr = [1, 2, 3, 4, 5];
$key1 = 1;
$key2 = 2;
echo $arr($key1).' '.$arr($key2);
?>

<?php
$arr = ['a' => 1, 'b' => 2, 'c' => 3];
$key = 'b';
echo $arr($key);
?>
```
