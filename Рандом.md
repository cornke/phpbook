### Случайное число
```php
<?php
echo mt_rand(1, 100);
?>

<?php
for ($i = 0; $i <= 10; $i++) {
    $arr[$i] = mt_rand(1, 100);
}
print_r($arr);
?>
```

### Перемешивание массивов
```php
<?php
$arr = [1, 2, 3, 4, 5];
shuffle($arr);
print_r($arr);
?>

<?php
$arr = range(0, 25);
shuffle($arr);
print_r($arr);
?>

<?php
$arr = range('a', 'z');
shuffle($arr);
print_r($arr);
?>
```

### Перемешивание строк
```php
<?php
echo str_shuffle('pppppppppperemeshivai v chem problema');
?>

<?php
$arr = range('a', 'z');
shuffle($arr);
echo substr(implode('', $arr), 0, 6)
?>
```
