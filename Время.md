### Работа с timestamp
```php
<?php
echo mktime(20, 50, 03, 03, 03, 2026);
?>

<?php
echo mktime(0, 0, 0, 01, 03, 2025);
?>

<?php
echo mktime(12, 59, 59, 12, 31);
?>

<?php
$date = mktime(13, 12, 59, 15, 15, 2001);
$today = time();
echo $today - $date;
?>

<?php
$date = mktime(7, 23, 48);
$today = time();
echo $today - $date;
?>
```

### Форматирование даты и времени
```php
<?php
echo date('H:i:s d.m.Y', time());
?>

<?php
echo date('Y-m-d', time());
?>

<?php
echo date('d.m.Y', time());
?>

<?php
echo date('d.m.Y H:i:s', time());
?>

<?php
echo date('d.m.Y', mktime(0, 0, 0, 02, 12, 2025));
?>

<?php
$week = ['Sunday', 'Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday','Saturday'];
$dayIndex = date('w', time());

echo $week[$dayIndex];
?>

<?php
$month = ['dec', 'jan', 'feb', 'mar', 'apr', 'may', 'jun','jul', 'aug', 'sep', 'oct', 'nov'];
$dayIndex = date('n', time());

echo $month[$dayIndex];
?>

<?php
$days = date('d', mktime(0, 0, 0, date('n') + 1, 0));
echo $days;
?>
```

### Разбор дат в свободном формате
```php
<?php
echo date('d.m.Y', strtotime('2025-12-31'));
?>

<?php
$currentDate = date('d');
$currentDay = date('w');
$nextMonday = 8-$currentDay;
echo date('d.m.Y', mktime(0, 0, 0, 0, $currentDate+$nextMonday));
?>

<?php
echo date('m.d.Y', strtotime('first day of next month'));
?>

<?php
echo strtotime('2023-12-25');
?>
```

### Изменение дат
```php
<?php
$date = '2025-12-31';
$dateArr = date_create($date);
date_modify($dateArr, '+2 day');
echo date_format($dateArr, 'Y-m-d');
?>

<?php
$date = '2025-12-31';
$dateArr = date_create($date);
date_modify($dateArr, '+3 days 1 month');
echo date_format($dateArr, 'Y-m-d');
?>

<?php
$date = '2025-12-31';
$dateArr = date_create($date);
date_modify($dateArr, '+1 year');
echo date_format($dateArr, 'Y-m-d');
?>

<?php
$date = '2025-12-31';
$dateArr = date_create($date);
date_modify($dateArr, '-3 day');

?>

<?php

$dateArr = date_create($date);

?>
```

### Сравнение timestamp с моментами
```php
<?php
$date_start = '2020-11-30';
$date_finish = '2020-12-01';
$date1 = explode('-', $date_start);
$date2 = explode('-', $date_finish);
$time1 = mktime(0,
        0, 0, $date1[1], $date1[0],
        $date1[2]);
$time2 = mktime(0,
        0, 0, $date2[1], $date2[0],
        $date2[2]);
if ($time1 > $time2) {
    echo 'first bigger';
} else {
    echo 'second bigger';
}
?>
```
