### Цикл и return
Автор выводит sum в цикле, это значит, что return отработает 1 раз. Хотели вывести сумму чисел от 1 до 5
```php
<?php
function func($num) {
    $sum = 0;

    for ($i = 1; $i <= $num; $i++) {
        $sum += $i;
    }
    return $sum;
}

echo func(5);
?>
```

### Применение return в цикле
```php
<?php
function func($num) {
    $res = $num;
    $i = 1;

    while (true) {
        $res /= 2;
        if ($res <= 10) {
            return $i;
        }
        $i++;
    }
}
echo func(25);
?>
```

### Советы по созданию функций
Можно исправить название функции на среднее арифметическое массива
```php
<?php

    $res = 0;
    foreach ($arr as $elem) {
        $res += $elem;
    }
    return $res / count($arr);
}
$numbers = [7, 9, 1, 0, -10];
var_dump(arrayArithmeticMean($numbers));
?>
```

Название не отражает суть функции, функция делает много действий
