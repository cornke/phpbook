### Основы работы с формами
```php
<form action="/result.php">
    <input name="AAA">
    <input age="34">
    <input salary="2000">
    <input type="submit">
</form>
```

### Метод отправки формы
```php
<form action="/result.php" method="GET">
    <input name="name">
    <input name="age">
    <input name="salary">
    <input type="submit">
</form>

<form action="/result.php" method="POST">
    <input name="name">
    <input name="age">
    <input name="salary">

</form>
```
