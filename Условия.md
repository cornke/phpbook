### Логичсекое И
```php
<?php

    echo '+++';
} else {
    echo '---';
}
?>

<?php

    echo '+++';
} else {
    echo '---';

    echo '+++';

    echo '---';
}
?>
```

### Группировка условий
```php
<?php
$num = 3;

if (($num > 5 and $num < 10) or $num == 20) {
    echo '+++';
} else {
    echo '---';
}
?>


<?php
$num = 3;

if ($num > 5 or ($num > 0 and $num < 3)) {
    echo '+++';
} else {
    echo '---';
}
?>

<?php
$num = 3;

if ($num == 9 or ($num > 10 and $num < 20) or ($num > 20 and $num < 30)) {
    echo '+++';
} else {
    echo '---';
}
?>
```
