### Конструкция Switch-case
```php
<?php

    case 1:
        echo "winter";
        break;
    case 2:
        echo "spring";
        break;
    case 3:
        echo "summer";
        break;
    case 4:
        echo "fall";
        break;
}
?>
```

### Конструкция match
```php
<?php
$season = 3;

$res = match ($season) {
    1 => 'winter',
    2 => 'spring',
    3 => 'summer',
    default => "fall",
};

echo $res;
?>
```
