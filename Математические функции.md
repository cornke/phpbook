### Степень и корень
```php
<?php
$a = 2;
echo pow($a, 10);
?>

<?php
$a = 245;
echo sqrt($a);
?>

<?php
 $arr = [4, 2, 5, 19, 
  13, 0, 10]; 
$res = 0;
foreach ($arr as $elem) {
$res += pow($elem, 2);
}
echo sqrt($res);
?>
```

### Функции округления 
```php
<?php
 echo round(sqrt(379));
?>

<?php
$a = sqrt(587);
$arr = [];

$arr["ceil"] = ceil($a);
$arr["floor"] = floor($a);
```

```php
?>
```

### Математические операции над массивами 
```php

?>

<?php

 echo array_product($arr);
?>

<?php
 $arr = [1, 2, 3, 4, 5];
       $devide = count($arr);
```
